Classic character building, modern clarity.

The Forge is a desktop character builder inspired by classic eTools workflows. It supports D and D 3.5e and 5e with a local-first data model, custom rule support, and zero required cloud accounts.

Why it exists

The Forge is built for players and DMs who want clarity, speed, and full control over their data. It keeps character sheets, rules, and tools together in one offline-friendly workspace.

Core pillars

Character builder

  • Rules-aware stats, saves, and combat grids
  • Skills, feats, bio, and portrait management
  • Export-ready sheets and summaries

Rules system

  • Alignment meters and edition-aware math
  • Custom rulesets and content packs
  • Local JSON import and export

Local AI helper

  • Optional, offline by default
  • Ask edition-aware questions
  • Pull from active character context

Workflow snapshot

1) Create or import a character - 2) Configure rules and progression - 3) Track changes and export
